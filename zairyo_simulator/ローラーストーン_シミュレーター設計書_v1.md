## 1. プロジェクト概要

### 1-1. 目的

ローラーストーン（床）の現場単位での材料費算出ツール。
施工面積を入力すると、工程ごとに必要な材料・数量・費用・㎡単価を自動算出する。

### 1-2. ユーザー

- ぬーまん自身（セメントアーティスト）
- 全国のローラーストーン加盟店（数十社、約100名）
- ※将来的に販売/サブスクリプション提供を想定した商用プロダクト

### 1-3. ロードマップ

| バージョン | 内容 |
|-----------|------|
| **v1.0（本設計書）** | 材料費シミュレーター（床のみ）。㎡入力→工程別材料費算出 |
| v1.5 | ローラーウォール（壁）を統合。1アプリ内で床/壁切替 |
| v2.0 | 見積書作成機能 |
| v3.0 | 在庫管理 + 現場別データ保存 |
| v4.0 | 認証・課金（買い切り or サブスク） |

### 1-4. デザイン方針

- CimentArtシミュレーターの設計思想をベースに、ローラーストーン仕様に置き換え
- ローラーストーン公式サイト（rollerstone.com）のブランドカラーに準拠
  - メインカラー: ピンク/マゼンタ系（ロゴカラー）+ ブラック
- スマホファースト（現場操作を前提）
- 価格表示: 税抜/税込の切替可能

### 1-5. CimentArtとの構造的な違い

| 項目 | CimentArt | ローラーストーン |
|------|-----------|----------------|
| 工程の決まり方 | 仕上げ材×施工箇所で工程が変わる | 基本1パターン。オプション（下地補修等）で一部増減 |
| 材料計算の単位 | 材料ごとにトータル数量 | **工程ごとに材料・数量を個別算出**し、積み上げで総合計 |
| 仕上げ材の選択 | 6種類から選択 | 選択なし（ローラーストーンは1製品） |
| 造形パターン | なし | 乱形・タイル・木目等（材料は同じ、技法のみ違い） |
| テープ使用 | なし | 乱形張りのみRSテープ使用 |

### 3-2. 施工工程と㎡あたり材料使用量

**ローラーストーン（床）の標準施工フロー:**

```
工程1: 高圧洗浄（オプション）
工程2: 下地補修・調整（オプション）→ 必要時、リキッド水塗布が入る
工程3: テープ貼り（乱形張りのみ）
工程4: リキッド水塗布
工程5: セメント打設（造形）
工程6: タフミルク水塗布（目地色下地）
工程7: 目地カラー塗布（着色）
工程8: 石色塗り（着色）
工程9: コート仕上げ
```

#### 工程別 材料使用量一覧

| 工程 | 材料 | 配合・希釈 | 使用量/㎡ | 算出ロジック |
|------|------|----------|---------|------------|
| **工程4: リキッド水塗布** | RSリキッド（原液） | 原液:水 = 1:1 | 原液 約1L/15㎡ | 原液1L+水1L=2L → 15㎡塗布 |
| **工程5: セメント打設** | インスタントセメント | - | 12.5kg/2.5㎡（=5kg/㎡） | 1練り=2.5㎡想定 |
| | RSプラス | セメントに混合 | 400g/2.5㎡（=160g/㎡） | 1練りあたり400g |
| | RSメイド | セメントに混合 | 800g/2.5㎡（=320g/㎡） | 1練りあたり800g |
| | RSリキッド（打設用） | ペール缶から柄杓取り | 約4〜5L/2.5㎡ | 柄杓1杯≒1L、1練り4〜5杯 |
| **工程3: テープ貼り** | RSテープ | - | 1巻/4㎡ | 乱形張りのみ。50巻/箱≒200㎡ |
| **工程6: タフミルク水** | RSタフミルク | タフミルク:水 = 1:1 | 原液 約1L/15㎡ | マニュアル記載値 |
| **工程7: 目地カラー** | RS基準色カラー（メイン） | そのまま or 調合 | 1缶(20kg)/100㎡ | メインの着色材 |
| | RSカラー | 石色・白ぼかし等 | メインの約1/4の消費量 | メイン4:RSカラー1の割合 |
| **工程8: 石色塗り** | （工程7と同じ材料） | - | 工程7に含む | |
| **工程9: コート仕上げ** | RSコート | 水:コート = 9:1 | コート原液 約1L/20㎡ | マニュアル記載値 |

#### 重要な注記（アプリ上に必ず表示）

- ※1練り（セメント12.5kg）= 約2.5㎡を想定。現場条件により前後します。
- ※推奨打設厚み: 2mm〜4mm
- ※RSリキッドは工程4（リキッド水塗布）と工程5（打設）で別々に消費。工程別に個別算出。
- ※着色材（RS基準色・RSカラー・ブレンドカラー）の消費量は色・デザインにより変動します。
- ※インスタントセメントの価格はエリアにより変動（¥700〜800/25kg袋）。

#### RSリキッドの使用構造（重要）

```
RSリキッド 1缶（20kg ≒ 20L）
  │
  ├── 工程4: リキッド水塗布用
  │     原液:水 = 1:1 で希釈
  │     原液1L → 15㎡
  │
  └── 工程5: セメント打設用
        ペール缶に水16〜17L + 原液2.5L = リキッド水 約19〜20L
        ペール缶1缶 → 4〜5練り分（柄杓4〜5杯/練り、柄杓1杯≒1L）
        RSリキッド1缶 → ペール缶約8回分

  ※各工程で使用するRSリキッドは別々に計算し、合算しない。
  ※工程別に必要量を表示し、総トータルは合計欄で表示。
```

| オプション | 条件 | 追加材料 |

## 5. 計算ロジック

### 5-1. 自動計算の流れ

```
入力: 施工面積(sqm), 造形パターン, テープ種類, 基準色, オプション
  │
  ├── 工程4: リキッド水塗布
  │   RSリキッド原液 = sqm / 15 (L)
  │   → RSリキッド缶数 = ceil(必要L数 / 20)
  │
  ├── 工程5: セメント打設
  │   練り回数 = ceil(sqm / 2.5)
  │   セメント = 練り回数 × 12.5kg → ceil(総kg / 25) 袋
  │   RSプラス = 練り回数 × 400g → ceil(総g / 9000) セット
  │   RSメイド = 練り回数 × 800g → ceil(総g / 20000) 袋
  │   RSリキッド(打設用) = 練り回数 × 4.5L(中間値) → ペール缶回数 → 原液L数
  │     ペール缶回数 = ceil(練り回数 / 4) ※1ペール缶=4〜5練り、安全側で4
  │     原液必要量 = ペール缶回数 × 2.5L
  │     → RSリキッド缶数 = ceil(原液L数 / 20)
  │
  ├── 工程3: テープ貼り（乱形張りのみ）
  │   テープ巻数 = ceil(sqm / 4)
  │   → テープ箱数 = ceil(巻数 / 50)
  │
  ├── 工程6: タフミルク水塗布
  │   RSタフミルク原液 = sqm / 15 (L)
  │   → RSタフミルク缶数 = ceil(必要L数 / 18)
  │
  ├── 工程7+8: 着色（目地カラー + 石色）
  │   RS基準色カラー = sqm / 100 → ceil(缶数)
  │   RSカラー = 基準色の1/4消費 → sqm / 400 → ceil(缶数)
  │
  └── 工程9: コート仕上げ
      RSコート原液 = sqm / 20 (L)
      → RSコート缶数: 20kg缶 or 4kg缶から最適選択
```

### 5-2. RSリキッドの合計表示

工程4と工程5で別々に算出したRSリキッドは、それぞれの工程カードで個別に表示する。
合計ボックスでは「RSリキッド（リキッド水塗布用）」「RSリキッド（打設用）」と分けて表示し、
発注用の総缶数も表示する。

### 5-3. 手動調整モード

- 各工程カードに「✏️ 手動で調整」ボタン
- 手動モードでは数量を自由に変更可能（**0も入力可 = 在庫対応**）
- 「↩ 自動に戻す」で元の自動計算値に復帰

### ローラーウォール（壁）統合時の変更点

ローラーウォールはローラーストーンと材料構成が異なる:
- RSモールド 5kg + RSカラー(基準色) 1.5L + 寒水石 3.75kg が1バッチ
- タフミルクは使用しない
- 目地色を先に全面塗布してからテープ貼り（ローラーストーンと逆）
- RSモールド（S/N/L）は気温帯で使い分け
- リキッド水の配合比が異なる（水4:リキッド1）

→ v1.5で壁タブを追加する際は、材料マスター・工程ロジック・計算式を壁専用で別定義する。

### タイル調の練りの違い

基本の乱形張りとタイル調では練りの配合が変わる可能性あり。
v1.0では基本配合（12.5kg/2.5㎡）で統一し、将来的に造形パターン別の配合分岐を追加。
