## 実装してほしいこと（v1.0 全機能）

### 1. 入力エリア

以下の入力欄を設ける:

- **お客様名**（テキスト）
- **現場名**（テキスト）
- **住所**
  - 郵便番号入力（7桁）→ 都道府県・市区町村を自動反映
  - 郵便番号APIは `https://zipcloud.ibsnet.co.jp/api/search?zipcode=` を使用
  - 番地・建物名は手動入力
- **施工面積**（数値入力 ㎡）
  - 寸法計算モードも付ける（L×Wを複数スペース分入力→合計㎡を自動集計）
- **造形パターン選択**（乱形張り / タイル調 / 木目調 / その他）
  - 乱形張り選択時のみ → テープ種類選択（Type1太幅/細幅、Type2太幅/細幅、Type3）
- **基準色選択**（以下から1つ）
  - ミルクティー(¥19,800) / イエローベージュ(¥19,800) / ピンクブラウン(¥19,800) / グレー(¥19,800)
  - オレンジ(¥21,000) / モスグリーン(¥21,000) / ブラック(¥21,000)
  - グレージュ(¥19,800)
- **オプション選択**（チェックボックス）
  - 高圧洗浄（有/無）→ 材料費には影響なし
  - 下地補修（有/無）→ 材料費には影響なし
  - 事前下地付着処理（有/無）→ RSメイド+RSリキッド追加
- **「材料費を算出する」ボタン**

### 2. 工程別材料費自動計算

**重要: CimentArtとは計算構造が異なる。材料ごとではなく「工程ごと」に材料と使用量を個別算出し、積み上げで総合計を出す。**

施工面積(sqm)を入力後、以下の各工程で必要な材料・数量・費用を算出:

#### 工程7: RSリキッド水塗布
```
RSリキッド原液 = sqm / 15 (L)
必要缶数 = ceil(必要L数 / 20)  ※1缶=20kg≒20L
費用 = 缶数 × ¥14,900
```

#### 工程8: セメント打設（メイン工程）
```
練り回数 = ceil(sqm / 2.5)  ※1練り=約2.5㎡

インスタントセメント:
  必要kg = 練り回数 × 12.5
  必要袋数 = ceil(必要kg / 25)  ※25kg/袋
  費用 = 袋数 × ¥750

RSプラス:
  必要g = 練り回数 × 400
  必要セット = ceil(必要g / 9000)  ※9kg/セット
  費用 = セット数 × ¥11,900

RSメイド:
  必要g = 練り回数 × 800
  必要袋数 = ceil(必要g / 20000)  ※20kg/袋
  費用 = 袋数 × ¥9,500

RSリキッド（打設用）:
  ペール缶回数 = ceil(練り回数 / 4)  ※1ペール缶=4〜5練り分、安全側で4
  原液必要量 = ペール缶回数 × 2.5 (L)
  必要缶数 = ceil(原液L数 / 20)
  費用 = 缶数 × ¥14,900
```

#### 工程6: RSテープ貼り（乱形張りのみ）
```
if 造形パターン === "乱形張り":
  必要巻数 = ceil(sqm / 4)
  必要箱数 = ceil(巻数 / 50)  ※50巻/箱
  費用 = 箱数 × ¥45,000
```

#### 工程12: RSタフミルク水塗布
```
RSタフミルク原液 = sqm / 15 (L)
必要缶数 = ceil(必要L数 / 18)  ※1缶=18kg
費用 = 缶数 × ¥21,600
```

#### 工程13+15: 着色（目地カラー + 石色塗り）
```
RS基準色カラー（メイン）:
  必要缶数 = ceil(sqm / 100)
  費用 = 缶数 × 選択した基準色の価格

RSカラー（白ぼかし・石色用）:
  消費量 = メインの約1/4 → sqm / 400
  必要缶数 = ceil(sqm / 400)  ※最低1缶
  費用 = 缶数 × ¥16,700
```

#### 工程16: RSコート仕上げ
```
RSコート原液 = sqm / 20 (L)

サイズ最適化:
  20kg缶(¥26,500) と 4kg缶(¥8,200) から最安の組合せを自動選択
```

#### オプション: 事前下地付着処理
```
if 事前下地付着処理 === true:
  RSメイド追加分 + RSリキッド追加分（数量は固定ではなく要現場判断）
  → アプリ上は「要現場判断」と表示し、手動入力可能にする
```

### 3. RSリキッドの表示ルール（重要）

**工程7と工程8で使うRSリキッドは別々に表示する。**

- 工程7のカードに「RSリキッド（リキッド水塗布用）○L」
- 工程8のカードに「RSリキッド（打設用）○L」
- 合計ボックスでは「RSリキッド 発注合計: ○缶」と統合表示も入れる

### 4. 手動調整モード

- 各工程カードに「✏️ 手動で調整」ボタン
- 手動モードでは数量を自由に変更可能
- **0個（0缶/0袋）の入力を許可する**（在庫で対応するケース）
- 「↩ 自動に戻す」で元の自動計算値に復帰

### 5. 結果表示

#### サマリーカード（3列）
- 材料費合計
- ㎡単価（材料費合計 ÷ 施工面積）
- 施工面積

#### 工程別材料カード
各工程ごとにカード形式で表示:
- 工程名・工程番号
- 材料名・荷姿・数量・小計
- 配合比・注記
- 「✏️ 手動で調整」ボタン

#### 合計ボックス
- 工程別の小計一覧
- 材料費合計
- ㎡単価
- 税抜/税込の切替ボタン

### 6. 材料単価表パネル

- ヘッダーに「単価表を見る」ボタン
- スライドアップパネルで全RS材料の価格・容量一覧を表示
- 税抜/税込を合計ボックスの切替と連動

### 7. 注記表示（アプリ上に必ず表示）

以下の注記をアプリ上の適切な位置に表示:
- ※1練り（セメント12.5kg）= 約2.5㎡を想定。現場条件により前後します。
- ※推奨打設厚み: 2mm〜4mm
- ※RSリキッドは工程ごとに個別算出しています。
- ※着色材の消費量は色・デザインにより変動します（目安値）。
- ※インスタントセメントの価格はエリアにより変動（¥700〜800/25kg袋）。
